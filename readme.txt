== Description ==
Радиоплеер для сайдбара

Задача плагина - обеспечить лёгкую установку радиоплеера.

* Плагин элементарно устанавливается и требует минимум настроек.
* Функции прослушивания радио становятся доступны сразу после установки.

Плагин разработан и поддерживается порталом Radiola.Audio.

**Основные функции**

* Обеспечение прослушивания до 5 различных радиостанций
* Настройка ширины под любой сайдбар
* Компактная и полная версия плеера
* Отображение названий проигрываемых песен
* Настраиваемый слайдер

После установки настройки плагина доступны через виджеты.
